Drop your plot photos in THIS folder.

How to use them:
1. Put your photos here, e.g. omr-cover.jpg, omr-2.jpg
   (lowercase names, no spaces).
2. In src/data/lands.js, reference them WITHOUT the word "public", like:

   images: [
     '/plots/omr-cover.jpg',   <- first one is the cover photo
     '/plots/omr-2.jpg',
   ],

That's it. The photos appear on the site automatically.
You can delete this README.txt file whenever you like.
